Condicionales

Los condicionales en python nos permiten ejecutar una o varias instrucciones si se cumple una condición. La palabra clave de este tipo de estructura es "IF" la cual se traduce al español como "SI"(de condición, no de afirmación)

Estructura de control IF

La condición es una expresión lógica que debe ser evaluada con el resultado de verdadero o falso. Si la condición se cumple, la instrucción se ejecutará.
